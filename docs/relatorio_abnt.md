# Relatório Técnico — Projeto Final com Pandas

## 1. Introdução

Este trabalho tem como objetivo aplicar os conceitos estudados na disciplina, utilizando Python e Pandas para leitura, transformação, análise e exportação de dados a partir da base Adult Income.

## 2. Objetivo

Desenvolver um fluxo de análise de dados estruturado, com foco em organização do projeto, tratamento da base, análise estatística descritiva e apresentação de resultados.

## 3. Base de dados

A base utilizada foi a Adult Income, disponibilizada pela professora para o desenvolvimento do projeto final.

## 4. Metodologia

A metodologia adotada será organizada em etapas:
- preparação do ambiente;
- leitura dos dados;
- inspeção inicial;
- transformação;
- análise exploratória;
- geração de relatórios;
- exportação dos resultados.
