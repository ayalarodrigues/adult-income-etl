# Adult Income ETL e Análise com Pandas

Projeto final da disciplina, desenvolvido com Python e Pandas, utilizando a base Adult Income.

## Objetivo
Construir um pipeline inicial de análise de dados, seguindo as etapas de organização, leitura, transformação, análise descritiva e geração de relatórios.

## Estrutura do projeto
- `data/raw`: dados brutos
- `data/processed`: dados tratados e exportados
- `src`: código-fonte
- `docs`: documentação e relatório

## Tecnologias utilizadas
- Python
- Pandas
- Git e GitHub

## Como executar
1. Criar e ativar o ambiente virtual
2. Instalar as dependências:
   ```bash
   pip install -r requirements.txt

## Organização adotada
O projeto foi estruturado desde o início com separação entre dados brutos, dados processados, código-fonte e documentação, para facilitar a manutenção, a análise e o versionamento no GitHub.

## Estratégia de desenvolvimento
