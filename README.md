# Adult Income ETL e Análise com Pandas

Projeto final da disciplina, desenvolvido com Python e Pandas, utilizando a base Adult Income.

## Objetivo
Construir um pipeline inicial de análise de dados, seguindo as etapas de organização, leitura, transformação, análise descritiva e geração de relatórios.

## Estrutura do projeto
- `data/raw`: dados brutos
- `data/processed`: dados tratados e exportados
- `src`: código-fonte
- `docs`: documentação e relatório

## Tecnologias utilizadas
- Python
- Pandas
- Git e GitHub
